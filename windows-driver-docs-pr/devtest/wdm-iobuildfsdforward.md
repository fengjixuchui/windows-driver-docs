# IoBuildFsdForward rule (wdm)


The **IoBuildFsdForward** rule specifies that a completion routine must be set before a driver calls [**IoCallDriver**](https://msdn.microsoft.com/library/windows/hardware/ff548336) or [**PoCallDriver**](https://msdn.microsoft.com/library/windows/hardware/ff559654) if the IRP is generated by a call to [**IoBuildAsynchronousFsdRequest**](https://msdn.microsoft.com/library/windows/hardware/ff548310).

|              |     |
|--------------|-----|
| Driver model | WDM |
