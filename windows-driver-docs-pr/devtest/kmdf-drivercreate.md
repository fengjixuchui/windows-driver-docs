# DriverCreate rule (kmdf)


The **DriverCreate** rule specifies that a driver that uses Kernel Mode Driver Framework (KMDF) must call the [**WdfDriverCreate**](https://msdn.microsoft.com/library/windows/hardware/ff547175) method to create a framework driver object from within its [**DriverEntry**](https://msdn.microsoft.com/library/windows/hardware/ff540807) routine.

|              |      |
|--------------|------|
| Driver model | KMDF |
