# PagedCode rule (wdm)


The **PagedCode** rule specifies that the driver calls the [**PAGED\_CODE**](https://msdn.microsoft.com/library/windows/hardware/ff558773) macro only when it is executing at **IRQL &lt;= APC\_LEVEL**.

|              |     |
|--------------|-----|
| Driver model | WDM |
