# !rsdt


The **!rsdt** extension displays the ACPI Root System Description Table.

Syntax

```dbgcmd
!rsdt 
```

## <span id="ddk__rsdt_dbg"></span><span id="DDK__RSDT_DBG"></span>


### <span id="DLL"></span><span id="dll"></span>DLL

Kdexts.dll
