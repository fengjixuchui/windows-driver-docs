# DEVPKEY_DrvPkg_BrandingIcon


The DEVPKEY_DrvPkg_BrandingIcon device property represents a list of icons that associate a device instance with a vendor.

Remarks
-------

A branding icon can be specified as an .ico file or as a resource within an executable file.

The format of an icon list is the same as that described for the [**DEVPKEY_DrvPkg_Icon**](devpkey-drvpkg-icon.md) device property.

You can set the value of DEVPKEY_DrvPkg_BrandingIcon by an [**INF AddProperty directive**](https://docs.microsoft.com/windows-hardware/drivers/install/inf-addproperty-directive) that is included in the [**INF *DDInstall* section**](https://docs.microsoft.com/windows-hardware/drivers/install/inf-ddinstall-section) of the INF file that installs a device. You can retrieve the value of DEVPKEY_DrvPkg_BrandingIcon by calling [**SetupDiGetDeviceProperty**](https://docs.microsoft.com/windows/desktop/api/setupapi/nf-setupapi-setupdigetdevicepropertyw).
