# Dropped Frame Properties


The [PROPSETID\_VIDCAP\_DROPPEDFRAMES](https://docs.microsoft.com/windows-hardware/drivers/stream/propsetid-vidcap-droppedframes) property set contains properties related to the dropping of video frames during a capture operation.
