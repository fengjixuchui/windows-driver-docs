# !ob, !od, !ow


## <span id="ddk__o__dbg"></span><span id="DDK__O__DBG"></span>


The **!ob**, **!od**, and **!ow** extension commands are obsolete. Use the [**ob, od, ow (Output to Port)**](ob--ow--od--output-to-port-.md) commands instead.
