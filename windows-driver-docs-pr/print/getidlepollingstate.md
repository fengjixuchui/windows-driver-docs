# GetIdlePollingState


The GetIdlePollingState command returns the state of the TCPMON idle polling.

The following tables shows the relationship between the return values and polling states.

<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th>Return value</th>
<th>Polling state</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>True</p></td>
<td><p>Enabled</p></td>
</tr>
<tr class="even">
<td><p>False</p></td>
<td><p>Disabled</p></td>
</tr>
</tbody>
</table>
