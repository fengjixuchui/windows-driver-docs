# OID_GEN_CO_XMIT_PDUS_OK

The OID_GEN_CO_XMIT_PDUS_OK OID specifies the number of PDUs transmitted without errors.

## Requirements

**Version**: Windows Vista and later
**Header**: Ntddndis.h (include Ndis.h)
