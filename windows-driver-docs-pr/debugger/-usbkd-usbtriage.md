# !usbkd.usbtriage


The **!usbkd.usbtriage** command displays a list of USB drivers and device objects.

```dbgcmd
!usbkd.usbtriage
```

## <span id="DLL"></span><span id="dll"></span>DLL


Usbkd.dll
