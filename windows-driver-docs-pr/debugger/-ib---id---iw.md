# !ib, !id, !iw


## <span id="ddk__i__dbg"></span><span id="DDK__I__DBG"></span>


The **!ib**, **!id**, and **!iw** extension commands are obsolete. Use the [**ib, id, iw (Input from Port)**](ib--iw--id--input-from-port-.md) commands instead.
