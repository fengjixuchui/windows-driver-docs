# !ecs


## <span id="ddk__ecs_dbg"></span><span id="DDK__ECS_DBG"></span>


The **!ecs** extension is obsolete. To edit the PCI configuration space, use [**!ecb**](-ecb---ecd---ecw.md), **!ecd**, or **!ecw**.
