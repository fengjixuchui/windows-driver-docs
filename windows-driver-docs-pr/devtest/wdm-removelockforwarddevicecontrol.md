# RemoveLockForwardDeviceControl rule (wdm)


The **RemoveLockForwardDeviceControl** rule verifies that calls to [**IoAcquireRemoveLock**](https://msdn.microsoft.com/library/windows/hardware/ff548204) and [**IoReleaseRemoveLock**](https://msdn.microsoft.com/library/windows/hardware/ff549560) are used correctly when the driver uses [**IoCallDriver**](https://msdn.microsoft.com/library/windows/hardware/ff548336) to forward an IRP to a another device.

|              |     |
|--------------|-----|
| Driver model | WDM |
